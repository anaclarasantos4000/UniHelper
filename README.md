# UniHelper

## Descrição

O UniHelper é um sistema desenvolvido em Shell Script para auxiliar estudantes na organização de tarefas acadêmicas e automatização de atividades do dia a dia no Linux.

## Objetivo

Criar um ambiente simples e intuitivo para organizar disciplinas, arquivos, backups e outras tarefas acadêmicas utilizando apenas o terminal.

## Tecnologias

- Bash (Shell Script)
- Linux
