# UniHelper

## Descrição

O UniHelper é um sistema desenvolvido em Shell Script (Bash) para auxiliar estudantes na organização de seus materiais acadêmicos em ambiente Linux.

O sistema permite criar e gerenciar disciplinas automaticamente, organizando os arquivos em uma estrutura padronizada de diretórios, facilitando o armazenamento de PDFs, anotações, trabalhos, provas e outros materiais.

Além disso, o projeto utiliza Git e GitHub para controle de versões durante o desenvolvimento.

## Objetivo

Criar um ambiente simples e intuitivo para organizar disciplinas, arquivos, backups e outras tarefas acadêmicas utilizando apenas o terminal.

## Funcionalidades

- Gerenciar Disciplinas
- Buscar Arquivos
- Novo Trabalho
- Informações do Sistema
- Sair

Ao selecionar Gerenciar Disciplinas, será possível:

- Criar uma nova disciplina;
- Acessar disciplinas;
- Listar disciplinas;
- Remover disciplinas;

## Como Executar 
1. Clone o repositório:
git clone https://github.com/anaclarasantos4000/UniHelper.git/
2. Entre na pasta do projeto:
cd ~/UniHelper
3. Dê permissão aos scripts:
chmod +x scripts/menu.sh
chmod +x scripts/funcoes.sh
4. Execute o programa:
./scripts/menu.sh
